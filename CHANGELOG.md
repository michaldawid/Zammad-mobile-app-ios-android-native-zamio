# Changelog

All notable changes to Zamio will be documented in this file.

## [1.0.0] - 2026-03

### Added
- Ticket management: browse overviews, create, edit, search
- Conversations: read, reply, internal notes, attachments
- Native push notifications via relay server
- Multi-language support (EN, DE, PL, FR, ES, PT, NL, UA)
- Dark mode (automatic)
- Multiple Zammad server support
- Secure credential storage (iOS Keychain / Android Keystore)
